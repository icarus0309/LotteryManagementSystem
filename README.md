# LotteryManagementSystem
 该管理系统分别针对普通用户、超级管理员、普通管理员、公证员建立，使用文件IO进行信息交互。用户可以登录、注册、购买彩票、查看中奖纪录、充值、注销；超级管理员账户名锁死但密码可修改，可以管理普通管理员和公证员的创建、修改、删除；普通管理员可以针对用户进行增、删、改、查、排；公证员可以进行发行彩票和开奖操作。奖项分为一等奖、二等奖、三等奖、特别奖，其中一等奖表示用户购买彩票数字数值与顺序全对、二等奖与三等奖表示只有部分数字顺序和数字为对、特别奖的含义是用户所购买的数字全部正确但顺序不是全对。当公证员开奖后，如果用户中奖且用户处于未登录状态(因为没有使用网络编程，所以难以实时更新)，那么下次用户登录时则会收到中奖提醒。
