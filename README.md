# LotteryManagementSystem
 该管理系统分别针对普通用户、超级管理员、普通管理员、公证员建立，使用文件IO进行信息交互。

用户可以登录、注册、购买彩票、查看中奖纪录、充值、注销；

超级管理员账户名锁死但密码可修改，可以管理普通管理员和公证员的创建、修改、删除；

普通管理员可以针对用户进行增、删、改、查、排；

公证员可以进行发行彩票和开奖操作。

奖项分为一等奖、二等奖、三等奖、特别奖，其中一等奖表示用户购买彩票数字数值与顺序全对、二等奖与三等奖表示只有部分数字顺序和数字为对、特别奖的含义是用户所购买的数字全部正确但顺序不是全对。

此外还需注意：

1.用户应当同时拥有ID和用户名，ID永远不重复，用户名可以在账户注销后另起一个相同的用户名

2.奖励设置：
	3.1 三等奖：中一个数，奖励1倍彩票单价 中奖概率：0.9*0.9*0.3 = 27.3%--> 27.3%
	3.2 二等奖：中两个数，奖励5倍彩票单价 中奖概率：0.1*0.1*0.9*3=2.7%--> 13.5%
	3.3 一等奖：中三个数，奖励100倍彩票单价 中奖概率:0.1%--> 10% 
	3.4 特别奖：三个数字全对，但位置与一等奖不一致 奖励50倍彩票单价 中奖概率 0.5%,特别奖可以与三等奖叠加-->10%
	期望收益率：1-0.273-0.135-0.1-0.1=39.2%

3.公证员开奖后，管理员发行下一期彩票前，用户没有彩票可以购买，如果用户中奖且用户处于未登录状态(因为没有使用网络编程，所以难以实时更新)，那么下次用户登录时则会收到中奖提醒。
